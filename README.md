# Job Skills Analysis

This project analyzes job data to identify the most in-demand technical and non-technical skills across different business majors.

## Tools Used
- Excel
- Power Query
- Power BI

## What I Did
- Cleaned and transformed 32,000+ job records using Power Query
- Split job functions and skills into structured format
- Categorized skills into technical and non-technical groups
- Used pivot tables and visualizations to identify top skills

## Key Insights
- Identified the most demanded skills across different majors
- Highlighted differences between technical and non-technical roles
- Provided insights to support career planning and curriculum decisions
